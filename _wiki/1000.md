# 1000번문제
## 문제
> 두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오.<br/>
> 입력<br/>
> 첫째 줄에 A와 B가 주어진다. (0 < A, B < 10)<br/>
> 출력<br/>
> 첫째 줄에 A+B를 출력한다.<br/>

```java
import java.util.Scanner;

public class Main{
    public static void main(String args[]){
        int a,b;
        
        Scanner scanner = new Scanner(System.in);
        a = scanner.nextInt();
        b = scanner.nextInt();
        scanner.close();
        
        System.out.println(a+b);
    }
}
```
