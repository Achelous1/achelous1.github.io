# 고양이

## 문제
* 아래 예제와 같이 고양이를 출력하시오.

## 입력
* 없음.

## 출력
* 고양이를 출력한다

## 예제입력
```
```

## 예제출력
```
\    /\
 )  ( ')
(  /  )
 \(__)|
```

## 제출
### Java8
```java
public class Main {
    public static void main(String args[]){
        System.out.println("\\    /\\");
        System.out.println(" )  ( ')");
        System.out.println("(  /  )");
        System.out.println(" \\(__)|");
    }
}
```

## 문제의 요점
* 백슬래시`\`의 이스케이핑을 해야 제대로 출력이 된다.
